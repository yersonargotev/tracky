# Triage labels

Tracky uses these local status labels in Markdown issue metadata:

- `needs-triage` — maintainer needs to evaluate.
- `needs-info` — waiting on reporter/user clarification.
- `ready-for-agent` — fully specified and ready for an implementation agent.
- `ready-for-human` — needs human implementation or judgment.
- `wontfix` — will not be actioned.

For this repo's local issue files, labels are plain text under each issue's metadata block.
